# Pokedex

## Features
* **Get Pokemon list**
* **Get Pokemon details**
* **Add/Remove Pokemon to a favorite list**

## Installation
1. Clone the project `git clone git@github.com:jeromephan/izberg-test.git`
2. `npm install`
3. `npm start`
4. Visit `http://localhost:3000`

## Packages used
1. React
2. Redux
3. React-router
4. Axios

## Api used
[Pokeapi](https://pokeapi.co/)
